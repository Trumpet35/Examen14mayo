Para alojar (o compartir) un repositorio Git en GitHub, siga los siguientes pasos 
Paso 1: Resgistrar una cuenta de Github
El primer paso para comenzar con el alojamiento de Github es crear una cuenta personal. Visite la página de registro oficial para registrarse 

Paso 2: Crear un repositorio remoto en GitHub.
Después de registrarse, crear un home (un repositorio) en github para el repositorio Git que quieres compartir  

Paso 3: Conectar el directorio Git del proyecto con el repositorio remoto.
Una vez que hayas creado un reposotirio remoto para tu proyecto, tienes que vincular el directorio del proyecto
